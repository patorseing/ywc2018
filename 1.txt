ให้น้องเล่าประสบการณ์การทำงานเกี่ยวกับการทำหรือพัฒนาเว็บไซต์ (สามารถแปะลิงค์ผลงานที่เคยทำไว้ได้)

ขอแนะนำตัวสักนิดนึง หนูชื่อ ภัทร ชื่อจริงคือ ณภัทรชล ไทพาณิชย์
ภัทรมีความสนใจด้านการเขียนโปรแกรมและการพัฒนา เว็บมาตั้งแต่ ม.4 ปัจจุบันเรียนอยู่ ปี 4 ที่ คณะ information and communication technology (ICT) มหาวิทยาลัยมหิดล และชื่นชอบการเรียน web programming ตอน อยู่ ปี 2 เป็นอย่างมาก
ตอนที่เรียนอยู่ปี 2 ได้ใช้ HTML, PHP, JS, และ CSS จนคล่อง 
ตอนปิดเทอมขึ้นปี 3 ไม่ได้ไปฝึกงานแต่นึกพิเรนไปเช่า server ทำเว็บ โดย wordpress ซึ่งเขียน wordpress theam [1] ทำได้ 1 ปี ก็เลิกเช่า server ไป
แต่ยังไม่ค่อยมีประสบการณ์การทำงานจนกระทั้ง ได้ไปฝึกงานตอน ปิดเทอมขึ้นปี 4 เป็นการไปขอพี่ที่บริษัทไปทำงาน บริษัทนั้นเป็น บริษัท startup แถว พระราม 9 คือ Be ID Corporation Co., Ltd. พี่ๆ senior ที่บริษัทให้ทำ web [2] โคย library ที่ใช้คือ ReactJS มีส่วนในการทำงานนิดหน่อย ในบางส่วนของเว็บ และ dashbroad ซึ่งเป็น produce ของบริษัท (ใช้ ReactJS + Redux (รู้สึกตอนนี้เปลี่ยนเป็น MobX) + JSS ประมาณนี้) ตอนที่เริ่มฝึกงานนั้น ภัทร ไม่เคยใช้ library หรือ framework ตัวไหนเลย ไม่ว่าจะเป็น ReactJS, nodejs, angular, และอื่นเลย แต่ภัทรค่อนข้างเรียนรู้เร็ว เริ่มเรียนรู้ ReactJS ได้ประมาณ 1 สัปดาห์ และมีโอกาสได้ แตะ nodejs กับ scala อีกนิดหน่อย และได้ทำอีกหลายๆอย่าง เช่น swift (iOS application) ได้ฝึกงานประมาณ 2 เดือน
